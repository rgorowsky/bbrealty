# General Maintenance
* To generate node_modules folder in contact-backend & lambda-deploy folders
    * First - package.json file needs to exist
    * Then - run 'npm install'
* To generate package.json file
    * Run 'npm inint -y'
    * The -y automatically fills default values.  Tweak later as needed to make sure versions are compatible
